To compile and run the program , import the source code as well as  input txt file onto Eclipse 
Once you have done that , right click Run configurations of the java file and then go to arguments ,
type in the input txt you want to sort . 
run the program should give you the sorted output, repeat this progress for the other inputs.

To run it on command line, make sure input file and java source  file are on the same folder,
compile the java source file and run it  with appropriate input  name should give you the same output .
